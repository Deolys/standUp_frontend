Для начала работы с Vite:
1) Ознакомиться с https://vitejs.dev/guide/
2) Открыть терминал в папке проекта и написать npm create vite@latest (предварительно установить npm)
3) Задать название проекта. Далее выбираем Vanilla -> JavaScript
4) Перейти в созданную папку через команду cd ./"Название проекта"/
5) Ввести команду npm i
